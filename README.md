# dapr-k6

Small proof of concept of using k6 load testing with dapr

## Requirements
You will need the following intalled:
- [k6](https://k6.io/)
- [dapr](https://dapr.io/)
- npm/node

## Instructions
Start the dummy-service
1. Navigate to the dummy-service folder
2. Run npm install
3. Run npm run start 

Run the k6 test
1. Navigate to the k6 folder
2. Run npm run test
